## Introduction
PyEdit is a basic command-line text editor written in Python. It allows you to load and view text files, but currently lacks the ability to write and save changes. This is a work in progress, and your contributions are welcome! ğŸ¤

## Getting Started
### Clone the repository
git clone https://github.com/jean0t/pyedit.git

### Run the editor
python editor_cli.py

## Usage
* Load a file: editor_cli.py file.txt
* Navigate: use arrow keys or `j`/`k` to scroll
* Quit: `Ctrl+Q`
